# Инструкция по созданию структуры проекта

## Windows (Command Prompt)

Скопируйте и вставьте эту команду в командную строку:

```cmd
cd c:\Project\site_maxvich1 && mkdir src && mkdir src\app && mkdir src\components && mkdir src\components\ui && mkdir src\components\layout && mkdir src\components\sections && mkdir src\components\forms && mkdir src\lib && mkdir src\data && mkdir src\types && mkdir public && mkdir public\images && mkdir public\videos && mkdir public\images\portfolio && mkdir public\images\team && mkdir public\images\hero
```

## После создания папок

Скопируйте файлы вручную:

1. Создайте файл `src\app\globals.css`
2. Создайте файл `src\app\layout.tsx`
3. Создайте файл `src\app\page.tsx`
4. Создайте файл `src\lib\utils.ts`
5. Создайте файл `src\lib\constants.ts`

Содержимое файлов см. в следующих сообщениях.
